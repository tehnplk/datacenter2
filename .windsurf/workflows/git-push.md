---
description: Commit and push all changes to remote in one step
---

1. ตรวจสอบไฟล์ที่มีการเปลี่ยนแปลง
// turbo
2. รัน git status
```
git status
```

// turbo
3. Stage ไฟล์ทั้งหมด
```
git add -A
```

4. Commit พร้อม message (แก้ข้อความได้)
```
git commit -m "update"
```

// turbo
5. Push ไปยัง remote
```
git push
```
